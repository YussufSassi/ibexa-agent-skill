# Ibexa DXP agent skill

Gives your agent deep knowledge of [Ibexa DXP](https://www.ibexa.co/) so it can help you write code, generate migrations, configure multisite setups, and anything else you'd normally dig through the docs for.

Works with all editions (Headless, Experience, Commerce) and covers the PHP API, search, templating, SiteAccess config, Page Builder, PIM, commerce, permissions, events, workflows, and more.

## Install

```bash
claude plugin add https://github.com/YussufSassi/ibexa-agent-skill
```

or 

```bash
npx skills add YussufSassi/ibexa-agent-skill --skill ibexa-dxp
```

## Usage

Just describe what you need. The skill kicks in automatically on Ibexa projects.
